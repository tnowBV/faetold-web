# Bookinsanity font variants

This app uses **Bookinsanity** (D&D body text font) from [Solbera's D&D 5e Fonts](https://github.com/jonathonf/solbera-dnd-fonts).

You already have:
- `Bookinsanity.otf` – regular

To enable **italic** and **bold** text, add these files to this folder:

1. **BookinsanityItalic.otf**  
   Download: [Bookinsanity Italic.otf](https://github.com/jonathonf/solbera-dnd-fonts/raw/master/Bookinsanity/Bookinsanity%20Italic.otf)  
   Save or rename it as: `BookinsanityItalic.otf`

2. **BookinsanityBold.otf**  
   Download: [Bookinsanity Bold.otf](https://github.com/jonathonf/solbera-dnd-fonts/raw/master/Bookinsanity/Bookinsanity%20Bold.otf)  
   Save or rename it as: `BookinsanityBold.otf`
